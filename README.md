# 🚀 AR QR Menu

A modern, responsive React landing page for **AR QR Menu**, a smart digital restaurant solution that enables QR-based ordering and augmented reality food preview experiences.

---

## ✨ Features

- 📱 **QR Menu System** – Instant digital menu access via QR scan  
- 🍽️ **AR Food Preview** – Visualize food in augmented reality before ordering  
- 📊 **Analytics Dashboard** – Track customer engagement and menu interactions  
- 🎨 **Custom Branding** – Fully customizable restaurant UI  
- 🌍 **Multi-language Support** – Serve global customers  
- 📅 **Table Booking System UI** – Reservation flow integration concept  

---

## 🛠 Tech Stack

- ⚛️ React 19  
- ⚡ Vite (Fast build tool)  
- 🎨 CSS3 (Modern responsive styling)  
- 🎯 React Icons
